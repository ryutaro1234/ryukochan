授業内容

初心者にも、わかりやすく。 プロじゃない人にもわかりやすく。 作曲教材サイト。

作編曲の裾の尾を広げたい。 そうすることで、将来的には音楽業界の発展につながる。

既存のWebサイトは、わかりにくい。主に構造とグラフィック。 （例えば、楽譜が読めない人をグラフィックでサポートする。）

そこのプラットフォームがWeb。

インプットとアウトプットは違う。 インプットの手伝いをしたい。 だから、DAWのUIデザインとは違う。

最終的目標 作曲できない人にそのページを読んでもらって、作曲してもらう。 弟に頼むか。

既存のサイトと比較する。 ピアノロールで説明する。

15日

Safariでのanimationが動かない問題ですが、キーフレームの書き方に原因があったようです。
classの追加はうまくいっていたようなので、cssの方に問題があると考え、
最初から.show01を追加しリロードしたところ、アニメーションが表示されませんでした。
.show01{ animation: basic01 1s fowards; } @keyflams basic-01{ 0%{fill:#ffff7f ;} 100%{fill:none ;} }
そこで、50%{fill:red; }を追加したところ、黄色から赤への変化が確認できました。
.show01{ animation: basic01 1s fowards; } @keyflams basic-01{ 0%{fill:#ffff7f ;} 50%{fill:red; } 100%{fill:none ;} }
そこから
.show01{ animation: basic01 1s fowards; } @keyflams basic-01{ 0%{fill:#ffff7f ;} 30%{fill:#ffff7f ;} 100%{fill:none ;} }
とする事により、問題の解決となりました。 問題のページ: https://ryutaro1234.github.io/daw-world/contents/basic.html 在リンク切れ 新しいページ↓
